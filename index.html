FLOJO.js

Flojo.js, a very lightweight jQuery script for loading images lazily.

This script instructs the browser to delay the download of images. Thus, you get the strictly necessary images and prevent the server from sending a lot of unnecessary information.

This is how it works!

This document contains a string of images placed one under another. If you attach Flojo.js to your HTML documents, the browser will be instructed to download the images as visitors scroll down/up the page. In order to fully appreciate how it works, you can use your browser's development tools.
